Projection directory

Contains the following files:

i) project_mono_FORBILD.m and line_integrals.m:	used to compute analytic projection data of the FORBILD phantom.
ii) radon_ell.m: used to compute analytic projection data of phantoms specified using Matlab's syntax (see documentation for the phantom command). Courtesy of A. Faridani.
iii) soft_tissue_correction.m: performs a soft-tissue-correction on polyenergetic data to produce simulated monoenergetic data.
